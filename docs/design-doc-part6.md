## 7. 前端页面设计

### 7.1 页面清单

| 页面 | 路由 | 用途 |
|------|------|------|
| 配置向导 | `/config` | 首次启动，设置API Key和模型 |
| 看板首页 | `/dashboard` | 所有订单的生产进度一览 |
| 订单列表 | `/orders` | 全部订单，支持搜索/筛选 |
| 订单详情 | `/orders/:id` | 单个订单的详细信息和工序 |
| 创建订单 | `/orders/new` | 手动创建或拍照上传 |
| 客户管理 | `/customers` | 客户信息维护 |
| 产品管理 | `/products` | 产品信息维护 |
| 预警中心 | `/alerts` | 呆滞预警、逾期预警 |
