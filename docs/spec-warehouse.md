# 仓库管理 Agent — 详细规格说明书

> 基于 Move Hermes 设计文档 Phase 3 扩展

---

## 1. Agent 概述

**仓库管理 Agent** 是 Move Hermes 的第三个定制 Agent，专注于解决小企业仓库管理的盲区问题。

**核心问题：**
- 库存多少全靠脑子记，经常超卖或缺料
- 找不到东西，翻半天
- 不知道哪些物料快过期/呆滞
- 盘点一次要停工半天

**解决方案：**
通过拍照盘点 + AI 自动计数 + 库存预警，让仓库管理一目了然。

## 2. 功能规格

### 2.1 库存管理模块

#### 2.1.1 物料档案
- 物料编号（自动生成：MAT-0001）
- 物料名称、规格型号、单位
- 分类：原材料 / 半成品 / 成品 / 辅料
- 存放位置（货架号/区域号）
- 安全库存量（低于此值预警）

#### 2.1.2 入库管理
- 采购入库：关联采购订单，AI识别送货单
- 生产入库：关联生产订单，完工后自动入库
- 手工入库：直接录入数量和批次
- 拍照上传送货单/入库单

#### 2.1.3 出库管理
- 领料出库：关联生产工单
- 销售出库：关联销售订单
- 退货出库：供应商退货
- 扫码/拍照快速出库

### 2.2 库存盘点模块

#### 2.2.1 拍照盘点
- 工人拍摄货架照片
- AI 自动识别物料和数量
- 系统对比理论库存 vs 实际库存
- 差异自动生成盘点报告

#### 2.2.2 定期盘点
- 月度全盘 / 季度抽盘
- 按区域/分类分批盘点
- 盘点任务分配给指定人员
- 盘点结果自动更新库存

### 2.3 预警模块

#### 2.3.1 库存不足预警
- 低于安全库存自动标黄
- 低于最低库存标红
- 预警推送至企微/飞书群

#### 2.3.2 呆滞物料预警
- 超过 90 天无变动物料标黄
- 超过 180 天无变动物料标红
- 呆滞物料占比统计

#### 2.3.3 保质期预警（预留）
- 近效期物料提醒
- 过期物料自动冻结

### 2.4 报表模块

#### 2.4.1 库存汇总表
- 当前库存总量/总金额
- 按分类/区域统计
- 出入库趋势

#### 2.4.2 盘点差异报告
- 盘盈/盘亏明细
- 差异率统计
- 责任追溯

## 3. 数据流

```
┌─────────────────────────────────────────────────────────────┐
│                     库存流转                                  │
│                                                             │
│  采购订单 → 收货确认 → 入库 → 库存增加                       │
│  生产工单 → 领料 → 库存减少 → 完工 → 成品入库                │
│  销售订单 → 出库 → 库存减少 → 发货确认                       │
│                                                             │
│  拍照盘点 → AI识别 → 对比理论库存 → 生成差异报告             │
│                                                             │
│  库存 < 安全库存 → 预警 → 提醒采购                           │
│  物料 90天不动 → 呆滞预警 → 提醒处理                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 4. 与订单/财务 Agent 的集成

```
订单生产 → 领料出库 → 仓库扣减
    │
    └── 成品入库 → 财务库存资产增加
            │
            └── 发货出库 → 财务确认收入

采购 → 入库 → 财务确认应付
    │
    └── 库存增加 → 可用生产

呆滞预警 → 提醒处理 → 减少库存占用资金
```
