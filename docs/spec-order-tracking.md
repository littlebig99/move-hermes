## 1. Agent 概述

**订单跟踪 Agent** 是 Move Hermes 的第一个定制 Agent，专注于解决小企业订单生产过程中的"黑盒"问题。

**核心问题：**
- 订单在哪道工序？看不到
- 加急单被淹没在普通单中
- 工序卡住几天没人发现
- 完工了不通知发货

**解决方案：**
通过 AI 驱动的自动化数据录入 + 实时生产看板 + 智能预警，让订单全流程透明化。

## 2. 功能规格

### 2.1 订单创建模块

#### 2.1.1 手动录入
- 填写客户、产品、数量、单价、交货日期
- 支持从客户/产品下拉列表选择
- 自动生成交付编号（ORD-YYYYMMDD-NNN）
- 可标记为"加急"

#### 2.1.2 拍照识别录入
- 上传纸质订单照片
- AI 自动识别并填充表单
- 人工复核确认
- 识别结果保存到历史，提高后续准确率

#### 2.1.3 企微/飞书群录入
- 在指定群内发送订单照片
- 机器人自动识别并创建订单
- 识别结果推送到管理群确认
- 确认后自动进入生产队列

### 2.2 工序管理模块

#### 2.2.1 工序模板
- 预设常用工序：下料 → 加工 → 组装 → 质检 → 包装 → 发货
- 支持自定义工序名称和顺序
- 每个订单可有不同的工序流程

#### 2.2.2 工序分配
- 订单创建后自动分配第一道工序
- 每道工序指定负责人（可选）
- 支持工序跳过和回退

#### 2.2.3 工序更新
- 工人通过企微/飞书群拍照上报
- AI 自动识别并更新进度
- 管理员可在看板手动更新

### 2.3 生产看板模块

#### 2.3.1 概览统计
- 生产中 / 待开工 / 已完成 / 加急 数量
- 按客户统计订单分布
- 今日新增订单数

#### 2.3.2 进度矩阵
- 行：订单号 + 客户名
- 列：各工序状态
- 颜色编码：✅完成 🔄进行中 ⏳待开工 🔴呆滞

#### 2.3.3 排序与筛选
- 按优先级排序（加急单置顶）
- 按交货日期排序（临近交货的靠前）
- 按客户筛选
- 按状态筛选

### 2.4 预警模块

#### 2.4.1 呆滞预警
- 默认阈值：工序停留超过 2 天标黄，超过 3 天标红
- 可自定义阈值
- 预警列表显示：订单号、工序名、停滞天数

#### 2.4.2 交期预警
- 提前 3 天提醒即将逾期的订单
- 提前 1 天强烈提醒
- 已逾期订单置顶显示

#### 2.4.3 通知方式
- 看板内红色角标
- 企微/飞书群推送通知
- 可选邮件通知（预留）

## 4. 接口契约

### 4.1 订单创建

```python
# 请求
POST /api/orders
{
    "customer_id": 1,
    "product_id": 5,
    "quantity": 500,
    "unit_price": 2.5,
    "priority": "urgent",
    "notes": "客户要求带合格证",
    "delivery_date": "2026-07-01"
}

# 响应
{
    "id": 101,
    "order_no": "ORD-20260623-001",
    "status": "pending",
    "message": "订单创建成功"
}
```

### 4.2 工序状态更新

```python
# 请求
PUT /api/tasks/{task_id}/status
{
    "status": "completed",
    "photo_url": "/data/photos/task_101_2.jpg",
    "ai_confidence": 0.95,
    "ai_notes": "完成数量: 500件, 工人: 李师傅"
}

# 响应
{
    "task_id": 15,
    "status": "completed",
    "next_task": {"id": 16, "name": "质检", "status": "pending"}
}
```

### 4.3 呆滞预警查询

```python
# 请求
GET /api/tasks/stalled?days=2

# 响应
{
    "stalled_tasks": [
        {
            "task_id": 12,
            "order_no": "ORD-20260620-003",
            "task_name": "焊接",
            "stalled_since": "2026-06-21",
            "days": 3
        }
    ]
}
```
