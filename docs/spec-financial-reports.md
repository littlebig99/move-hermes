# 财务报表 Agent — 详细规格说明书

> 基于 Move Hermes 设计文档 Phase 2 扩展

---

## 1. Agent 概述

**财务报表 Agent** 是 Move Hermes 的第二个定制 Agent，专注于解决小企业财务管理的混乱问题。

**核心问题：**
- 每月对账需要 3-5 天
- 纸质单据散落各处，用途不明确
- 是否付款、有没有报销全要靠微信问一遍
- 不知道本月赚了多少钱

**解决方案：**
通过 AI 驱动的票据识别 + 自动分类 + 报表生成，将月度对账从 3-5 天缩短到 30 分钟。

## 2. 功能规格

### 2.1 票据管理模块

#### 2.1.1 票据录入
- 拍照上传发票/收据/合同
- AI 自动识别：金额、日期、供应商、品名、税额
- 自动分类：收入 / 支出 / 报销 / 预付
- 人工复核确认

#### 2.1.2 票据关联
- 票据自动关联到对应订单（通过金额/日期匹配）
- 手动关联：将多张票据归集到一个订单
- 未关联票据标记为"待归类"

#### 2.1.3 票据搜索
- 按日期范围搜索
- 按金额搜索
- 按关键词搜索（供应商名、品名）
- 按类型筛选（收入/支出/报销）

### 2.2 应收应付模块

#### 2.2.1 应收账款
- 从订单自动生成应收款项
- 状态追踪：未收 / 部分收款 / 已收 / 逾期
- 逾期提醒（超期 7 天标黄，30 天标红）
- 收款记录：拍照上传收款凭证

#### 2.2.2 应付账款
- 从采购/支出票据自动生成应付款
- 状态追踪：未付 / 部分付款 / 已付
- 到期提醒
- 付款记录：拍照上传付款凭证

### 2.3 报表生成模块

#### 2.3.1 月度利润表
- 收入汇总（按客户/产品/月份）
- 支出汇总（按类别/供应商/月份）
- 净利润 = 收入 - 支出 - 税费
- 同比/环比增长

#### 2.3.2 现金流表
- 本月现金流入
- 本月现金流出
- 净现金流
- 期末余额

#### 2.3.3 自定义报表
- 按时间段自定义
- 按客户/产品维度下钻
- 导出 CSV/Excel

### 2.4 报销管理模块

#### 2.4.1 报销申请
- 员工拍照上传报销票据
- AI 自动填写：金额、日期、事由
- 主管审批流程（简单：同意/驳回）
- 审批通过后计入支出

#### 2.4.2 报销统计
- 个人报销历史
- 部门报销汇总
- 月度报销趋势

## 4. 与订单跟踪 Agent 的集成

```
订单完成 → 触发应收款项
    │
    ├── 客户付款 → 关联应收 → 标记已收
    │
    └── 未付款 → 逾期预警 → 催款提醒

采购支出 → 关联订单/产品
    │
    ├── 供应商发票 → AI识别 → 计入支出
    │
    └── 付款记录 → 关联应付 → 标记已付

月度结账 → 生成利润表
    │
    ├── 总收入 = Σ 已收款 + 应收未收
    ├── 总支出 = Σ 已付款 + 应付未付
    └── 净利润 = 总收入 - 总支出
```
