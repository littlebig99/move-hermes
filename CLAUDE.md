# GStack Skill Routing

This project uses GStack for AI-assisted engineering.

## Skill Routing

When working on this project, load the appropriate GStack skill:

- **Requirements/Product**: `gstack:office-hours`, `gstack:plan-ceo-review`
- **Architecture/Design**: `gstack:plan-eng-review`, `gstack:plan-design-review`
- **Specification**: `gstack:spec`
- **Implementation**: `gstack:ship`, `gstack:review`
- **Debugging**: `gstack:investigate`
- **QA**: `gstack:qa`, `gstack:qa-only`

## Project Context

- **Product**: Move Hermes — U盘智能体 for small enterprises
- **MVP**: Order Tracking Agent
- **Tech Stack**: FastAPI + SQLite + Static HTML frontend
- **Target**: 20人以下制造企业
- **Deployment**: Portable USB drive, double-click to start
